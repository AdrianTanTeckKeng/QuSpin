Tools module (quspin.tools)¶

classes and tools to aid in calculations with QuSpin.

expm_multiply_parallel(A[, a]) This class warps some c++ code which implements the scipy.expm_multiply code with openmp.

block_tools¶

block_ops(blocks, static, dynamic, …[, …]) Split up the dynamics of a state over various symmetry sectors.
block_diag_hamiltonian(blocks, static, …) Block-diagonalises a Hamiltonian obeying a symmetry.

Floquet¶

Floquet(evo_dict[, HF, UF, thetaF, VF, n_jobs]) Calculates the Floquet spectrum, Floquet Hamiltonian and Floquet states.
Floquet_t_vec(Omega, N_const[, len_T, N_up, …]) Creates a Floquet time vector with fixed number of points per period.

measurements¶

ent_entropy(system_state, basis[, …]) Calculates entanglement entropy of a subsystem using Singular Value Decomposition (svd).
diag_ensemble(N, system_state, E2, V2[, …]) Calculates expectation values in the Diagonal ensemble of the initial state.
ED_state_vs_time(psi, E, V, times[, iterate]) Calculates the time evolution of initial state using a complete eigenbasis.
obs_vs_time(psi_t, times, Obs_dict[, …]) Calculates expectation value of observable(s) as a function of time in a time-dependent state.
project_op(Obs, proj[, dtype]) Projects observable onto symmetry-reduced subspace.
KL_div(p1, p2) Calculates Kullback-Leibler divergence of two discrete probability distributions.
mean_level_spacing(E) Calculates the mean-level spacing of an energy spectrum.
evolve(v0, t0, times, f[, solver_name, …]) Implements (imaginary) time evolution for a custom user-defined first-order ODE.
